# Windows virtueller Desktop auf Surface

## Einführung

Mit dem Windows-virtuellen Desktop auf Surface können Sie die Virtual Desktop Infrastructure (VDI) auf einem Surface-Gerät ausführen, das die Linien zwischen dem lokalen Desktop und dem virtuellen Desktop verwischt, wobei sich die Fingereingabe-, Stift-, Freihand-und biometrische Authentifizierung sowohl auf physische als auch auf virtuelle Umgebungen erstrecken. Als weiteren Meilenstein in der Entwicklung des Computings werden in Windows Virtual Desktop auf Surface <a href="#1"><sup> 1 </sup></a> Microsoft 365-virtualisierte in der Azure-Cloud mit erweiterten Sicherheitsschutz Funktionen, Verwaltbarkeit auf Unternehmensebene und erweiterte Produktivitätstools von Windows 10 auf Surface kombiniert.  Diese Kombination aus Premium-Formfaktoren und virtueller Desktop-Infrastruktur in Azure bietet außergewöhnlichen Kunden nutzen für Benutzererlebnisse, Portabilität, Sicherheit, Business Continuity und modernes Management.

### Windows Virtual Desktop

Windows Virtual Desktop (Oktober) ist ein umfassender Desktop-und App-Virtualisierungs-Service, der in der Azure Cloud ausgeführt wird. Es ist die einzige virtuelle Desktopinfrastruktur, die vereinfachte Verwaltung, Multi-Session Windows 10, Optimierungen für Office 365 ProPlus und Unterstützung für Remote Desktop Dienste (RDS)-Umgebungen bietet. Mit Oktober können Sie Windows-Desktops und-Apps auf Azure schnell bereitstellen und skalieren sowie integrierte Sicherheits-und Compliance-Funktionen nutzen.

### Integrationen von Windows Virtual Desktop-Partnern

Eine Liste der genehmigten Partner Anbieter und unabhängigen Softwarelieferanten für Windows Virtual Desktop finden Sie unter [Integrationen von Windows Virtual Desktop-Partnern](https://docs.microsoft.com/azure/virtual-desktop/partners). Einige Partner stellen auch Virtual Desktop as a Service (daas) bereit. DaaS befreit Sie davon, ihre eigenen virtuellen Maschinen (VMS) zu verwalten, indem Sie einen vollständig verwalteten, Turnkey-Desktop-und-Virtualisierungs-Service bereitstellen. Die Möglichkeit, maßgeschneiderte Desktops an Benutzer überall in der Welt zu liefern, ermöglicht es Unternehmen, sich schnell an sich verändernde Marktbedingungen anzupassen, indem Cloud-Desktops bedarfsgerecht gedreht werden – wann und wo Sie benötigt werden.

### Offline-und Geräte Zugriff für mehr produktive Erfahrungen

Üblicherweise funktionieren VDI-Lösungen nur, wenn der Endpunkt mit dem Internet verbunden ist. Was passiert aber, wenn das Internet oder die Stromversorgung aus irgendeinem Grund nicht verfügbar ist (aufgrund von Mobilität, auf einem Flugzeug oder Stromausfälle usw.)?
 
Um die Geschäftskontinuität zu unterstützen und die Mitarbeiter produktiv zu halten, können Surface-Geräte die virtuelle Desktopoberfläche mit dem Offlinezugriff auf Dateien, Microsoft 365 und Anwendungen von Drittanbietern problemlos erweitern. Herkömmliche apps wie Microsoft Office, verfügbar über. x86, x64, universelle Windows-Plattform, Arm-Plattformen, ermöglichen Benutzern, im Offlinemodus produktiv zu bleiben. Dateien aus der virtuellen Desktop Cloud-Umgebung können auch lokal auf Surface mithilfe von OneDrive for Business für den Offlinezugriff synchronisiert werden. Sie können die Gewissheit haben, dass alle lokal "zwischengespeicherten" Informationen auf dem neuesten Stand und sicher sind.

Zusätzlich zum Hinzufügen von Unterstützung für den Offlinezugriff auf apps und Dateien sind Surface-Geräte so konzipiert, dass Sie kollaborative Erfahrungen wie Microsoft Teams "auf dem Gerät" optimieren. Obwohl einige VDI-Lösungen die Verwendung von Teams über eine virtuelle Sitzung unterstützen, können Benutzer von der optimierten Funktionalität profitieren, die von einer lokal installierten Instanz von Teams bereitgestellt wird. Durch das Lokalisieren von Kommunikations-und Zusammenarbeits-Apps für Multimedia-Kanäle wie sprach-, Video-und Live Beschriftungen können Unternehmen die Möglichkeiten von Surface Devices optimal nutzen, um optimierte Microsoft 365-Erfahrungen bereitzustellen. Das Aufkommen von Surface Artificial Intelligence (AI) oder "AI-on-Device" bringt neue Funktionen zum Leben, wie beispielsweise Eye-Look-Technologie, die das Aussehen Ihrer Augen anpasst, damit das Publikum sieht, wie Sie bei der Kommunikation per Video direkt auf die Kamera schauen.
 
Eine Alternative zur lokalen Installation herkömmlicher Anwendungen besteht darin, die neueste Version von Microsoft Edge zu nutzen, die Unterstützung für Progressive Web Apps (PWA) bietet. PWAs sind nur Websites, die schrittweise verbessert werden, um wie Native apps auf unterstützenden Plattformen zu funktionieren. Die Qualitäten einer PWA kombinieren die besten Web-und systemeigenen Apps durch zusätzliche Features wie Push-Benachrichtigungen, Aktualisieren von Hintergrunddaten, Offline-Unterstützung und vieles mehr.

### Virtuelle GPUs

GPUs eignen sich hervorragend für AI-Rechen-und grafikintensive Arbeitslasten und helfen Kunden, Innovationen durch Szenarien wie Highend-Fernvisualisierung, Deep Learning und Predictive Analytics zu fördern.  Dies ist allerdings nicht ideal für Profis, die Remote oder unterwegs arbeiten müssen, da unterschiedliche Grade interner GPU-Leistungsstärke an die physischen Geräte gebunden sind, wodurch Mobilität und Flexibilität eingeschränkt werden.
 
Zur Lösung dieses Azure bietet die N-Serie von virtuellen Computern mit NVIDIA GPU-Funktionen (vGPU). Mit vGPUs können Sie die GPU-Leistung auf mehreren virtuellen Computern freigeben oder anspruchsvolle Arbeitslasten übertragen, indem Sie einem einzelnen virtuellen Computer mehrere GPUs zuweisen. Für Surface bedeutet dies, dass unabhängig davon, welches Gerät Sie verwenden, von der höchst tragbaren Oberfläche 2 bis zum schlanken und stylischen Surface Laptop 3, Ihr Gerät Zugriff auf eine leistungsfähige Grafikleistung von Server-Class bietet.  Mit Oberflächen-und vGPUs können Sie alle Ihre liebsten Oberflächen kombinieren, um Stift-, Touch-, Tastatur-, Trackpad-und PixelSense-Displays zu verwenden, wobei die Grafikfunktion nur in Hochleistungs-Computing-Umgebungen zur Verfügung steht. 
 
Azure N-Serie bringt diese Funktionen auf Ihrem Surface-Gerät zum Leben, sodass Sie auf beliebige Art und Weise arbeiten können – egal, wo Sie sich gerade befinden.  [Weitere Informationen zu Azure N-Series-und GPU-optimierten Größen für virtuelle Maschinen.](https://docs.microsoft.com/azure/virtual-machines/sizes-gpu)

### Sicherheit und Verwaltung

Von Chip zu Cloud, Microsoft 365 und Surface unterstützt Organisationen dabei, geschützt und auf dem neuesten Stand zu bleiben.
Da sowohl Surface-Hardware als auch Software von Microsoft entworfen, erstellt und getestet wurde, können die Benutzer davon überzeugt sein, dass Sie produktiv und durch führende Technologien von Chip zu Cloud geschützt sind.  Mit einer erhöhten Anzahl von Benutzern, die Remote arbeiten, ist der Schutz von Unternehmensdaten und geistigem Eigentum wichtiger als je zuvor. Der virtuelle Windows-Desktop ist auf einem Zero-Trust-Sicherheitsmodell konzipiert, in dem jede Zugriffsanforderung stark authentifiziert, innerhalb von Richtlinieneinschränkungen autorisiert und auf Anomalien überprüft wird, bevor der Zugriff gewährt wird.
 
Durch die Maximierung der Effizienz des Cloud-Computings kann das moderne Management die Anforderungen von Benutzern, Stakeholdern und Kunden in einem zunehmend wettbewerbsorientierten Geschäftsumfeld besser erfüllen. So können Sie beispielsweise Surface-Geräte mit minimaler Interaktion von Ihrem Team in Betrieb nehmen. Das Setup erfolgt automatisch und selbst gewartet. Updates sind sowohl für Ihr Team als auch für Ihre Benutzer schnell und problemlos. Sie können Geräte unabhängig von ihrem physikalischen Standort verwalten.
 
Zu den Sicherheits-und Verwaltungsfunktionen, die mit Windows Virtual Desktop auf Surface bereitgestellt werden, gehören:

- **WindowsUpdate.** Wenn Sie Windows auf dem neuesten Stand halten, können Sie den neuen Sicherheitsbedrohungen immer einen Schritt voraus sein. Windows 10 wurde von Grund auf für mehr Sicherheit entwickelt und nutzt die neuesten Hardwarefunktionen, um die Sicherheit zu verbessern. Mit einem speziell erstellten UEFI <a href="#5"><sup> 5- </sup></a> und Windows Update für Unternehmen, das auf sich entwickelnde Bedrohungen reagiert, ist der End-to-End-Schutz sicher und vereinfacht.

- **Hardware Verschlüsselung.** Mit der Geräteverschlüsselung können Sie die Daten auf der Oberfläche schützen, sodass nur autorisierte Personen darauf zugreifen können. Alle Surface for Business-Geräte verfügen über ein diskretes Trusted Platform Module (dTPM), das Hardware geschützt ist, während Software geschützte Schlüssel und Messungen verwendet, um die Software Gültigkeit zu überprüfen.
- **WindowsDefender.** Windows Defender Antivirus vereint Maschinelles Lernen, Analyse von großen Daten, detaillierte Bedrohungs Resistenz Forschung und die Microsoft-Cloud-Infrastruktur zum Schutz von Geräten.  Das Tool ist integriert und benötigt keine zusätzlichen Agents, die auf Geräten oder in der VDI-Umgebung bereitgestellt werden können, wodurch die Verwaltung vereinfacht und die Inbetriebnahme des Geräts optimiert wird. Windows Defender ist integriert und benötigt keine zusätzlichen Agents, die auf dem Gerät oder in der VDI-Umgebung bereitgestellt werden, wodurch die Verwaltung vereinfacht und die Geräteinbetriebnahme optimiert wird. Die echte out-of-the-Box-Erfahrung.
- **Austauschbare Laufwerke** – eine Teilmenge der neueren Surface-Geräte verfügt über herausnehmbare SSD-Laufwerke <a href="#6"><sup> 6 </sup></a> , die eine bessere Kontrolle über die Datenaufbewahrung bieten.
- **Moderne Authentifizierung –** Microsoft 365 und Surface ist eine einheitliche Plattform, die alle Windows-Sicherheitsfeatures bereitstellt (vorbehaltlich Lizenzierung und Aktivierung). Alle Surface-Portfolio-Geräte werden mit einer maßgeschneiderten Kamera ausgeliefert, die für Windows Hello for Business entwickelt wurde und biometrische Sicherheit bietet, die nahtlos von On-Device-zu VDI-basierten Erfahrungen verbleibt.
- **Modernes Firmware-Management** – mithilfe der Geräte-Firmware-Konfigurationsschnittstelle (DFCI) <a href="#7"><sup> </sup></a> können 7 IT-Administratoren Hardwareelemente auf Firmware-Ebene wie Mikrofone, USB-Anschlüsse, SD-Kartensteckplätze, Kameras und Bluetooth Remote deaktivieren, wodurch die Stromversorgung des Peripheriegeräts aufgehoben wird. Die Windows Defender-Anmelde Informations Überwachung verwendet die Virtualisierungs-basierte Sicherheit, damit nur privilegierte System Software darauf zugreifen kann.
- **Abwärts-und Aufwärtskompatibilität** – Windows 10-Geräte bieten eine abwärts-und Aufwärtskompatibilität zwischen Hardware, Software und Diensten. Microsoft hat eine starke Vergangenheit mit der Beibehaltung der Legacy-Unterstützung von Hardware, Peripheriegeräten, Software und Diensten, während die neuesten Technologien integriert sind. Unternehmen können IT-Investitionen planen, um eine lange Nutzungsdauer zu haben.
- **Brücke für ältere Windows 7-Arbeitsauslastungen** – für Lösungsszenarien, die von Legacy-Windows-Betriebssystemumgebungen abhängig sind, können Unternehmen VDI-Instanzen von Windows 7 verwenden, die in Azure ausgeführt werden. Dies ermöglicht die Unterstützung auf modernen Geräten wie Surface ohne das Risiko, sich auf ältere Windows 7-Computer zu verlassen, die nicht mehr die neuesten Sicherheitsupdates erhalten.  Zusätzlich zu diesen Vorteilen für "zukünftige Korrekturen" wird die Migration aller älteren Arbeitslasten erheblich vereinfacht, wenn die moderne Windows 10-Hardware bereits bereitgestellt wird.
- **Zero-Touch-Bereitstellung** – Autopilot ist die empfohlene moderne Verwaltungs Bereitstellungsoption für Surface-Geräte. Windows Autopilot on Surface ist eine Cloud-basierte Bereitstellungstechnologie in Windows 10. Sie können Windows Autopilot auf Surface verwenden, um Geräte in einem Zero-Touch-Prozess Remote bereitzustellen und zu konfigurieren. Windows Autopilot-registrierte Geräte werden beim ersten Start über das Internet identifiziert, und zwar über eine eindeutige Geräte Signatur, die als Hardwarehash bezeichnet wird. Sie werden automatisch registriert und mit modernen Verwaltungslösungen wie Azure Active Directory (Azure AD) und Verwaltung mobiler Geräte konfiguriert.
